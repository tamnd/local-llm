The exhaustive reference for the `llmgw` gateway. The [CLI](/reference/cli/) page
lists every flag the binary defines and the HTTP routes it serves. The
[configuration](/reference/configuration/) page documents the full YAML schema:
the bind, auth, models, and aliases blocks, plus the in-process engine. The
[release notes](/reference/release-notes/) page explains how a version is tagged
and published.
